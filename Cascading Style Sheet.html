Cascading Style Sheet (CSS)

    Cascading Styles Sheet are stylesheet language that used to denote the presentation of a HTML or XML document. CSS describes how elements should display on screen or on other media. It is Developed by Wium Lee and Bert Boss in 1996.

    • CSS 1
      • Level of 1 cascading style sheet mechanism
      • Released on December 17, 1996
      • Now Obsolete
    • CSS 2.0
      • Developed by W3c
      • Released in May 1998
      • Capabilities like absolute type of coding
    • CSS 2.1
      • Removes poorly supported and not fully interoperable features in CSS 2.0
      • Adds already implemented browser extensions
      • Released on June 7, 2011
    • CSS 3
      • Divided into several documents called modules
      • Modules has its own capabilities or extends features defined in CSS 2
      • Released on June 19,2012
