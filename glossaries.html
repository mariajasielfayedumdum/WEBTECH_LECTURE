GLOSSARY OF TERMS

ACRONYMS

ACRONYMS MEANING DEFINITION
AJAX Asynchronous JavaScript and XML it is the use of the XMLHttpRequest object to communicate with server-side scripts. It can send as well as receive information in a variety of formats, including JSON, XML, HTML, and even text files.
CGI Common Gateway Interface offers a standard protocol for web servers to execute programs that execute like Console applications (also called Command-line interface programs) running on a server that generates web pages dynamically.
CSS Cascading Stylesheets is a computer language that is used to write formatting instructions ( rules ). These rules tell a web browser how webpage content should 'look'â€” in terms of: layout, position, alignment, width, height, etc.
CRLF Carriage Return Line Feed it's a DOS hangover from the olden days from when some devices required a Carriage Return, and some devices required a Line Feed to get a new line, so Microsoft decided to just make a new-line have both characters, so that they would output correctly on all devices.
DHTML Dynamic Hyper Text Markup Language is an umbrella term for a collection of technologies used together to create interactive and animated web sites by using a combination of a static markup language (such as HTML), a client-side scripting language (such as JavaScript), a presentation definition language (such as CSS), and the Document Object Model (DOM).
DOM Document Object Mode s an application programming interface (API) for valid HTML and well-formed XML documents. It defines the logical structure of documents and the way a document is accessed and manipulated.
EIS Enterprise Information Systems is any kind of information system which improves the functions of an enterprise business processes by integration.
HAML HTML Abstraction Markup Language A templating system to avoid writing the inline code in a web document and make HTML easy and clean.
HTML Hyper Text Markup Language a standardized system for tagging text files to achieve font, color, graphic, and hyperlink effects on World Wide Web pages.
HTTP Hyper Text Transfer Protocol an application layer communication protocol used to access resources (hypertext/hypermedia) on the world wide web (www).
HTTPS Hyper Text Mark-up Language Secure the protocol over which data is sent between your browser and the website that you are connected to.
LESS Less a dynamic style sheet language that can be compiled into Cascading Style Sheets (CSS) and run on the client side or server side.
MIME Multipurpose Internet Mail Extensions a standard for formatting files of different types, such as text, graphics, or audio, so they can be sent over the Internet and seen or played by a web browser or email application.
PHP Hypertext Preprocessor is an HTML-embedded Web scripting language.
RFC Request For Comments or Remote Function Call is a formal document from the Internet Engineering Task Force ( IETF ) that is the result of committee drafting and subsequent review by interested parties.
SASS Syntactically Awesome Style Sheets An extension of CSS that adds power and elegance to the basic language. It allows to use variables, nested rules, mixins, inline imports, and more.
SCSS Sassy CSS newer syntax of SASS that uses braces to denote code blocks and semicolons to separate lines within a block.
SVG Scalable Vector Graphics is an XML-based vector image format for two-dimensional graphics with support for interactivity and animation.
URI Uniform Resource Identifier is a string of characters used to identify a resource. Such identification enables interaction with representations of the resource over a network, typically the World Wide Web, using specific protocols.
URL Uniform Resource Locator the address of a World Wide Web page.
XHTML Extesible Hyper Text Mark-up Language is part of the family of XML markup languages. It mirrors or extends versions of the widely used Hypertext Markup Language (HTML), the language in which Web pages are formulated.
XML Extesible Mark-up Language a metalanguage which allows users to define their own customized markup languages, especially in order to display documents on the Internet.
XLink XML Linking Language is an XML markup language and W3C specification that provides methods for creating internal and external links within XML documents, and associating metadata with those links.
XSL Extesible Stylesheet Language a file that describes how to display an XML document of a given type. Itshares the functionality and is compatible with CSS2 (although it uses a different syntax).
